add `.env` file on root `metyou/.env` with format like this :

```
# clerk
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=

NEXT_PUBLIC_CLERK_SIGN_IN_URL=/sign-in
NEXT_PUBLIC_CLERK_SIGN_UP_URL=/sign-up

# stream
NEXT_PUBLIC_STREAM_API_KEY=
STREAM_SECRET_KEY=
```
